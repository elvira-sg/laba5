📁 Bank System - Windows Forms Application with SQLite Database

Описание проекта

Десктопное приложение для управления клиентами банка с использованием Windows Forms и SQLite базы данных. Проект реализован в рамках лабораторной работы №5 по программированию.

Функционал
    ✅ Управление клиентами (добавление, редактирование, удаление)
    ✅ Работа с SQLite базой данных
    ✅ Экспорт/импорт данных в файлы
    ✅ Сортировка по балансу
    ✅ Валидация вводимых данных
    ✅ Два режима работы: с БД и локальный

Требования
    .NET Framework 4.7.2 или выше
    Visual Studio 2019/2022
    System.Data.SQLite NuGet пакет

🚀 Установка и запуск

     1. Клонирование репозитория:
git clone https://github.com/your-username/BankSystem.git
cd BankSystem

    2. Открытие проекта в Visual Studio:
⦁	        Запустите Visual Studio 2019/2022
⦁	        Выберите "Open a project or solution"
⦁	        Найдите и откройте файл BankSystem.sln

     3. Восстановление NuGet пакетов:
⦁	        Правой кнопкой по решению в Solution Explorer
⦁	        Выберите "Restore NuGet Packages"
⦁	        ИЛИ используйте Package Manager Console: Update-Package -Reinstall

     4. Настройка базы данных:
⦁	        Проект автоматически создаст файл bank.db при первом запуске
⦁	        Файл БД будет создан в папке bin\Debug\

     5. Сборка и запуск:
⦁	        Нажмите F5 для запуска в режиме отладки
⦁	        ИЛИ Ctrl+F5 для запуска без отладки
⦁	        ИЛИ выберите Debug → Start Debugging

📖 Инструкция по использованию

    Таблица клиентов - отображает всех клиентов в формате:
        Имя
        Паспорт (8 символов)
        Баланс (в рублях)

    Панель управления:
        ✅ Использовать БД - переключатель режимов работы
        ➕ Добавить - открывает форму добавления клиента
        ✏️ Изменить - редактирование выбранного клиента
        ❌ Удалить - удаление выбранного клиента с подтверждением
        💾 Экспорт в файл - сохранение текущих данных в файл
        📥 Импорт из файла - загрузка данных из файла
        🔼 Сортировка по балансу - сортировка по возрастанию баланса
        🗃️ Экспорт БД в файл - сохранение всей БД в файл
        📂 Импорт в БД из файла - загрузка данных в БД из файла
